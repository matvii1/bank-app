# Bank application

## Description
The user is able to log in and register a new bank account. On the **payments** page he can transfer money to anybody who is registered on the platform.

## Stack / Features
 - React Native
 - Firebase auth, database
 - Chat support
 - Typescipt
 - Dayjs
 - Tailwind
 - Custom hooks
 - Fetch API
 - Material icons
